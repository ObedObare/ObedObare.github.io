# Lumina Studios Application Documentation

## 1. Project Overview
**Lumina Studios** is a premium photography and videography management application. It features a futuristic "Cyber-Studio" aesthetic (Glassmorphism) and serves as a digital portfolio for clients and a management terminal for administrators.

---

## 2. Core Features

### 2.1 For Clients
*   **Multi-Page Hub**: Navigate through Story, Archive, Services, Pricing, and Contact sections using a smooth fade-and-scale ViewPager.
*   **Advanced Booking**: A futuristic booking system with a custom calendar, time slots, and real-time cost calculation.
*   **Visual Archive**: A high-quality gallery categorized by service type (Portraits, Fashion, etc.).
*   **Social Integration**: Quick links to TikTok, Instagram, YouTube, and Facebook.
*   **Direct Contact**: One-tap calling to +254 143 356 401 and direct email access.

### 2.2 For Administrators
*   **Secure Admin Login**: Exclusive access for authorized operators.
*   **Admin Dashboard**: A central command center to:
    *   Manage Studio Services (Modules).
    *   Control Gallery Media (Archive).
    *   Track and manage Client Bookings.
*   **Smart Navigation**: Admins get a special "Admin Panel" button on the main gallery for quick access to the dashboard.
